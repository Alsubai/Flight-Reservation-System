Flight Reservation System

Project Name:

Flight Reservation System

Description:

A console-based program that allows users to:
	•	Book a new flight ticket
	•	View current reservation
	•	Cancel a reservation
	•	Exit the system

The program stores passenger’s name, flight number, and destination. Input validation ensures proper data entry.

Features:
	•	Add, view, and cancel a reservation
	•	Input validation for better user experience
	•	Simple and intuitive menu-driven interface

How to Run:
	1.	Compile the code with any C++ compiler.
	2.	Run the executable in the terminal/console.
	3.	Follow the menu instructions to use the system.
